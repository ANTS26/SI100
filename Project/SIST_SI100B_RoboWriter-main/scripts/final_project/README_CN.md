# 上海科技大学 SI100B 项目：RoboWriter (机械臂写字)

## 简介
这是 **上海科技大学 SI100B 项目：RoboWriter** 的自述文件。本文件解释了项目的要求和评分标准，包括项目实现、项目报告和项目展示。请通读本文件，并在开始项目前确保小组成员中有人使用 Windows 平台。

## I. 项目实现 (45 分)
### 要求
#### 基础要求
1.  **机械臂模型 (2 分)**：代码仓库最新版本中的 `universal_robots_ur5e` 模型：https://github.com/MAYBERoboLab/SIST_SI100B_RoboWriter
2.  **起始姿态和终止姿态 (5 分)**：机械臂可以从任意姿态开始，但在完成书写后需要停止在关节配置：$[0.0,-2.32,-1.38,-2.45,1.57,0.0]$ rad。
3.  **书写区域 (5 分)**：书写区域是平面 z=0.1 m 上的一个正方形。该正方形由其四个顶点定义：$(0.5, 0.1, 0.1)$, $(0.5, 0.6, 0.1)$, $(-0.5, 0.6, 0.1)$, 和 $(-0.5, 0.1, 0.1)$ m。
4.  **抬起末端执行器 (5 分)**：机械臂的末端执行器应在每一笔画和每一个字符之间抬起。
5.  **书写内容 (10 分)**：机器人应在书写区域内书写一名学生的姓名（可以在小组中选择一名）的中文和拼音，以及该学生的学号。实时可视化文字的脚本/笔画。
6.  **插值方法 (7 分)**：使用多种不同的插值方法来模拟真实的笔画。
7.  **书写速度 (3 分)**：机器人的书写速度应设置为合理的值，以便整个书写时间（从机器人的起始姿态到终止姿态）少于 3 分钟。
8.  **绘制关节状态曲线 (3 分)**：绘制机械臂在书写一个字时的所有关节状态，假设这个字是你的中文姓氏。
9.  **演示视频 (5 分)**：录制视频演示书写过程，你可以调整相机以获得更好的视角。视频名称应为“小组成员姓名 + Final Project Demo”，格式为 MP4。

#### 加分项
1.  **Git (1 分)**：使用 Git 管理代码，每个小组成员至少需要有 3 次提交。
2.  **书写任意单词 (2 分)**：给定任意字符串，你需要用机械臂书写字符串中的单词。字符串仅包含英文字母、空格、数字、逗号和句号。
3.  **在球面上书写 (3 分)**：在由 $(x-0)^2+(y-0.35)^2+(z-1.3)^2=0$ 和 $z\leq 0.1$ 定义的球体内表面区域上书写文字。

### 评分标准
1.  对于基础要求，如果你未能完成要求 $k+1$，你将只能获得从要求 1 到 $k (k=1,2,3,4,5,6,7,8)$ 的分数。如果你未能完成多个要求，$k$ 将是未完成的最小要求索引。
2.  对于基础要求 7，如果你只使用线性插值，你将获得 3 分。
3.  加分任务可以任意组合完成。
